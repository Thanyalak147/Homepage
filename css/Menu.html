บริการเทรนเนอร์ส่วนตัว

โปรแกรมเริ่มต้น (Beginner Program)

สำหรับผู้ที่เพิ่งเริ่มต้นออกกำลังกาย เทรนเนอร์จะออกแบบโปรแกรมที่เหมาะสมกับระดับความฟิตของคุณ รวมถึงการสอนท่าออกกำลังกายพื้นฐานและการใช้เครื่องออกกำลังกายอย่างถูกต้อง

Learn More

โปรแกรมลดน้ำหนัก (Weight Loss Program)

เทรนเนอร์จะช่วยวางแผนการออกกำลังกายและให้คำแนะนำด้านโภชนาการเพื่อช่วยคุณลดน้ำหนักอย่างมีประสิทธิภาพและปลอดภัย พร้อมติดตามผลและปรับโปรแกรมตามความเหมาะสม

Learn More

โปรแกรมเพิ่มกล้ามเนื้อ (Muscle Building Program)

สำหรับผู้ที่ต้องการเพิ่มกล้ามเนื้อ เทรนเนอร์จะออกแบบโปรแกรมการฝึกที่เน้นการยกน้ำหนักและการเสริมสร้างกล้ามเนื้อ รวมถึงให้คำแนะนำในการรับประทานอาหารเสริม

Learn More

คลาสโยคะ (Yoga Class)

เพิ่มความยืดหยุ่นและความสมดุลด้วยการฝึกโยคะในกลุ่มขนาดเล็ก โค้ชของเราจะนำการฝึกที่ช่วยลดความเครียดและปรับปรุงความยืดหยุ่นให้กับสมาชิก

Learn More

คลาสพิลาทิส (Pilates Class)

ฟื้นฟูและเสริมสร้างกล้ามเนื้อแกนกลางร่างกายด้วยการฝึกพิลาทิส คลาสนี้ช่วยเพิ่มความแข็งแรงและการทรงตัวโดยใช้การฝึกท่าและเทคนิคที่ถูกต้อง

Learn More

คลาสเต้น (Dance Class)

สนุกกับการเต้นในคลาสที่ออกแบบมาเพื่อเพิ่มความกระฉับกระเฉงและพลังงานในร่างกาย คลาสนี้เหมาะสำหรับทุกระดับและช่วยเพิ่มความสุขในแต่ละวัน
